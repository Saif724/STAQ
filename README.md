# STAQ

**STAQ (Smart Task Automation Queue)** is an intelligent task scheduling and workflow automation platform built with Go, React, PostgreSQL, and Redis.

## Features

- Task Scheduling
- Workflow Automation
- Queue Management
- Multi-worker Execution
- Retry Policies
- Execution History
- REST API
- Email Verification
- JWT Authentication

## Tech Stack

### Backend

- Go
- Gin
- PostgreSQL
- Redis

### Frontend

- React
- TypeScript
- Tailwind CSS
